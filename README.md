# 🍽️ The Bali-Umar – Premium Restaurant Website

A modern, fully responsive restaurant website built for **The Bali-Umar**.  
Designed with a premium yellow & black theme inspired by the brand’s PDF design.

This website allows customers to easily browse the menu and place orders directly through WhatsApp.

---

## 🚀 Live Features

- ✅ Premium hero banner design
- ✅ Fully responsive (Mobile + Tablet + Desktop)
- ✅ Modern yellow & black theme
- ✅ Easy-to-use menu layout
- ✅ One-click WhatsApp ordering
- ✅ Floating WhatsApp button
- ✅ Clean and simple user interface
- ✅ Fast loading & lightweight

---

## 📲 WhatsApp Ordering System

Customers can:

- Click **Order Now** button
- Click **Order** under any menu item
- Click floating WhatsApp icon

The website automatically:

- Detects device (Mobile/Desktop)
- Opens WhatsApp app (mobile)
- Opens WhatsApp Web (desktop)
- Prefills order message

No external APIs required.
